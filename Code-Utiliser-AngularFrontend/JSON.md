# JSON (JavaScript Object Notation)

## Qu'est-ce que c'est ?
JSON est un format de données léger utilisé pour échanger des données entre un client et un serveur.

## Structure de base d'un objet JSON

```json
{
    "nom": "John Doe",
    "age": 21,
    "ville": "Paris",
    "langages": ["HTML", "CSS", "JavaScript"],
    "estEtudiant": true
}
